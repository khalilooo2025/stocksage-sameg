Instructions
Format de fichier CSV :
  • Encodage : UTF-8 (ou Latin-1)
  • Delimiteur : Point-virgule (;) ou virgule (,)
  • Premiere ligne : En-tetes des colonnes
Exemple CSV Produits :
Code;Designation;Categorie;Unite;Prix Unitaire HT
CLIM-001;Climatiseur Split;Climatisation;pce;45000.00
CABLE-01;Cable 3x2.5;Electrique;m;150.00
Exemple CSV Clients :
Code;Nom;Contact;Email;Telephone;Adresse;Ville
CLI001;Entreprise ABC;M. Dupont;abc@mail.com;0555123456;Rue 1;Alger
Procedure d'import :
  1. Exportez d'abord vos donnees pour avoir le bon format
  2. Modifiez le fichier CSV dans Excel/LibreOffice
  3. Selectionnez le fichier et cliquez "Importer"
  4. Les codes existants seront mis a jour
  5. Les nouveaux codes seront ajoutes
Important : La colonne Code est obligatoire. Si le code existe deja, l'enregistrement sera mis a jour.
Astuce : Commencez par exporter pour obtenir le format exact, puis modifiez le fichier et re-importez.
